# Calorie Finder - Поиск калорийности продуктов

## О проекте

Calorie Finder - это десктопное приложение для поиска информации о пищевой ценности продуктов. Приложение использует открытую базу данных OpenFoodFacts и предоставляет удобный графический интерфейс на PyQt5.

## Возможности

- Поиск по штрихкоду - точный поиск продукта по штрихкоду
- Поиск по названию - поиск продуктов по текстовому запросу
- Детальная информация - полные данные о пищевой ценности
- Асинхронные запросы - неблокирующий интерфейс
- Удобный GUI - современный интерфейс на PyQt5

## Технологии

- Python 3.8+ - основной язык программирования
- PyQt5 - графический интерфейс
- Requests - HTTP-запросы к API
- OpenFoodFacts API - база данных продуктов

## Установка

### 1. Установка зависимостей

pip install -r requirements.txt

### 2. Запуск приложения

python calorie_finder_app.py

## Использование

### Поиск по штрихкоду
1. Перейдите на вкладку "Поиск по штрихкоду"
2. Введите штрихкод продукта (например: 5449000000996)
3. Нажмите кнопку "Найти" или Enter

### Поиск по названию
1. Перейдите на вкладку "Поиск по названию"
2. Введите название продукта (например: "творог 5%")
3. Выберите продукт из таблицы двойным кликом
